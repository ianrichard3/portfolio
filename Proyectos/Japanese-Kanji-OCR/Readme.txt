Evidentemente a aplicación no esta terminada. Pero es un proyecto WIP Que utilizando APIs permite sacar screenshot a kanji de la pantalla y realizar OCR, es decir extraer el texto. Para por ultimo, obtener el significado y lectura del mismo


Tecnología utilizada:
-Utilización de API de Jisho, y Convertio
-OCR (Character Recognition)
-Tomado de Screenshot
