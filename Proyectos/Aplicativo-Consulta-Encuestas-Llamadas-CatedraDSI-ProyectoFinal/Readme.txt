Este es un proyecto realizado como Trabajo practico final de la Catedra Diseño de sistemas de Información
De la carrera de Ingeniería de Sistemas de Información

Básicamente simula un  caso de uso de un Sistema EVR real.
Específicamente de la consulta de encuestas realizadas sobre las llamadas.


Conocimientos Empleados:
-Interfaz Grafica con Python
-Programación Orientada a Objetos
-Acceso a Base de Datos y mapeo de Objetos de Persistencia
-Realización de Casos de Uso
